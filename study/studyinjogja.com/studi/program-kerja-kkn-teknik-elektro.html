Berikut adalah contoh program kerja KKN Jurusan Teknik Elektro:

Tema: Meningkatkan Aksesibilitas dan Pemanfaatan Teknologi Elektro di Masyarakat

Hari 1-2:

  • Orientasi dan pengenalan lingkungan kerja
  • Pembagian tugas dan pembentukan kelompok kerja
  • Sosialisasi dengan masyarakat setempat
  • Survey kebutuhan dan potensi desa terkait teknologi elektro

Hari 3-4:

  • Kegiatan penyuluhan tentang teknologi elektro, seperti prinsip-prinsip kelistrikan, instalasi listrik, dan teknologi komunikasi
  • Kegiatan pelatihan keterampilan teknologi elektro, seperti pemrograman mikrokontroler dan perakitan alat elektronik
  • Kegiatan pemberdayaan masyarakat melalui pengembangan teknologi elektro

Hari 5-6:

  • Kegiatan pengembangan teknologi elektro di desa, seperti pemasangan jaringan listrik dan telekomunikasi, pembangunan pompa air tenaga surya, dan pengembangan energi terbarukan
  • Kegiatan pelatihan teknologi elektro untuk masyarakat setempat, seperti perbaikan peralatan listrik rumah tangga dan perakitan alat elektronik sederhana
  • Kegiatan pengembangan potensi teknologi elektro berbasis lingkungan

Hari 7-8:

  • Kegiatan pengembangan teknologi elektro berbasis industri, seperti pengembangan industri elektronik dan perangkat telekomunikasi
  • Kegiatan pelatihan teknologi elektronik dan perakitan alat elektronik industri
  • Kegiatan diskusi tentang pengembangan industri elektronik dan perangkat telekomunikasi di desa

Hari 9-10:

  • Kegiatan pengembangan teknologi elektro berbasis digital, seperti pengembangan aplikasi teknologi elektro dan platform internet of things (IoT)
  • Kegiatan pelatihan pemrograman dan pengembangan aplikasi teknologi elektro
  • Kegiatan pembuatan video pembelajaran tentang teknologi elektro dan aplikasi teknologi elektro

Hari 11-12:

  • Kegiatan pengembangan teknologi elektro di sektor pertanian, seperti pengembangan alat pengairan dan alat pertanian berbasis teknologi
  • Kegiatan diskusi tentang pengembangan potensi teknologi elektro di sektor pertanian
  • Kegiatan pengembangan teknologi elektro di sektor pariwisata, seperti pengembangan aplikasi pariwisata dan sistem manajemen pariwisata berbasis teknologi

Hari 13-14:

  • Kegiatan pengembangan teknologi elektro di sektor kesehatan, seperti pengembangan alat kesehatan dan perangkat medis berbasis teknologi
  • Kegiatan diskusi tentang pengembangan potensi teknologi elektro di sektor kesehatan
  • Kegiatan kampanye tentang pentingnya pengembangan teknologi elektro di berbagai sektor

Hari 15:

  • Evaluasi program kerja KKN dan penyusunan laporan
  • Penutupan dan perpisahan dengan masyarakat setempat
